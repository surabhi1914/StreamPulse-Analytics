# Data Sources

## Spotify Web API

Purpose:
Collect metadata and current account information.

Expected Data:

- User Profile
- Playlists
- Playlist Tracks
- Saved Tracks
- Top Artists
- Top Tracks

---

## Spotify Account Export

Purpose:
Collect historical listening activity.

Expected Data:

- Streaming History
- Artist Information
- Track Information
- Listening Timestamp
- Device Information

---

## Known Limitations

Unavailable:

- Internal Spotify Revenue
- Real Premium Conversions
- Global User Behavior
- Internal A/B Test Results
